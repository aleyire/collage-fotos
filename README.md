## Collage de imágenes

En este desafío se crea un servidor con Express y el paquete express-fileupload para almacenar las imágenes que estarán dentro del collage

### Materia aplicada

- Express-fileupload
- Postman y body-parser
- Eliminando archivos

### 
- npm init
- npm install -y
- npm install --save express-fileupload
- npm i express
